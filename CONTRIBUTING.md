# Katkıda Bulunma

1. Depoyu forkla.
2. Yeni bir dal oluştur: `git checkout -b feature/yeni-ozellik`
3. Değişikliklerini test et.
4. Açıklayıcı bir commit oluştur.
5. Pull request gönder.

Yeni içerikler harici lisanslı görsel veya ses kullanmamalıdır. Mobil görünüm, klavye erişilebilirliği ve çevrimdışı çalışma korunmalıdır.
